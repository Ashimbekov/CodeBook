# CodeBook — Платформа для изучения программирования

## Обзор

Веб-платформа для изучения языков программирования на Vue 3. Начинаем с Java — полный курс от основ до фреймворков (50+ модулей). Контент разжёвывается максимально понятно с аналогиями, множеством примеров и практическими заданиями.

## Целевая аудитория

Смешанная: от полных новичков до продвинутых. Контент от "что такое переменная" до фреймворков и многопоточности.

## Стек технологий

- **Vue 3** (Composition API) + **Vite**
- **Vue Router** — клиентский роутинг
- **Pinia** — стейт-менеджмент
- **highlight.js** — подсветка синтаксиса
- **localStorage** — хранение прогресса (MVP)

## Архитектура

Чистый SPA без бэкенда. Весь контент хранится в JS-файлах внутри проекта. Прогресс пользователя в localStorage с возможностью добавить серверную синхронизацию позже.

## Роутинг

| Путь | Страница | Описание |
|------|----------|----------|
| `/` | HomePage | Список языков (карточки курсов) |
| `/course/:lang` | CoursePage | Модули курса (карточки модулей) |
| `/course/:lang/module/:moduleId` | ModulePage | Уроки модуля (сайдбар + список) |
| `/course/:lang/module/:moduleId/lesson/:lessonId` | LessonPage | Контент урока или практика |

Ленивая загрузка: каждый модуль данных — отдельный чанк.

### Типы контент-блоков

| Тип | Описание | Визуал |
|-----|----------|--------|
| `text` | Обычный текст, аналогии | Стандартный параграф |
| `heading` | Подзаголовок внутри урока | h3 |
| `code` | Блок кода с подсветкой | Тёмный фон, кнопка копирования |
| `note` | Важная заметка | Жёлтый блок с иконкой 📝 |
| `warning` | Частая ошибка | Красный блок с иконкой ⚠️ |
| `tip` | Полезный совет | Зелёный блок с иконкой 💡 |
| `image` | Картинка/схема | Картинка с alt-текстом |
| `list` | Маркированный список | ul > li |

## Система прогресса

### Хранилище (Pinia + localStorage)

```js
{
  completedLessons: {
    'java:1:1': true,
    'java:1:2': true
  },
  completedTasks: {
    'java:1:practice-1': true
  }
}
```

### Что отслеживается

- Прочитанные уроки — кнопка "Прочитано" внизу урока
- Решённые задания — кнопка "Решил!"
- Прогресс по модулю — вычисляется: X из Y уроков
- Общий прогресс курса — вычисляется: процент завершённых модулей
- Кнопка "Сбросить прогресс" в настройках

### Отображение прогресса

- Главная: процент по курсу на карточке
- CoursePage: прогресс-бар на каждом модуле
- Сайдбар: галочки ✅ напротив пройденных уроков
- Активный урок выделен цветом

## Визуальный стиль

Светлая минималистичная тема:

- **Фон:** #f8f9fb (страницы), #fff (карточки, сайдбар)
- **Текст:** #1a1a2e (заголовки), #444 (основной), #999 (вторичный)
- **Акцент:** #667eea (ссылки, активный урок)
- **Прогресс:** #4caf50 (зелёный)
- **Практика:** #f5a623 (оранжевый)
- **Код:** тёмная тема (#1e1e2e фон) с подсветкой Dracula-стиль
- **Шрифт кода:** JetBrains Mono или системный monospace
- **Скругления:** 8-12px на карточках, 4px на кнопках
- **Тени:** мягкие (0 2px 8px rgba(0,0,0,0.06))

## Навигация

- **Главная:** карточки курсов (языков) с прогрессом
- **Страница курса:** карточки модулей с прогрессом
- **Внутри модуля:** сайдбар слева с деревом уроков, контент справа
- **Breadcrumbs:** CodeBook › Java › Модуль › Урок
- **Навигация внизу урока:** "← Предыдущий" / "Прочитано — Далее →"
- **Мобильная версия:** сайдбар в бургер-меню

## Практические задания

Формат самопроверки:

1. Пользователь видит задачу, требования, ожидаемый вывод
2. Решает у себя локально в IDE
3. Может раскрыть подсказку
4. Нажимает "Показать решение" — видит эталонный код + разбор
5. Нажимает "Решил!" — засчитывается в прогресс
6. Или "Не получилось — пройти теорию заново" — возврат к теории

Уровни сложности: Лёгкая (зелёный), Средняя (оранжевый), Сложная (красный).

## Авторизация (будущее, не MVP)

- На старте: всё в localStorage, кнопка "Войти" неактивна или скрыта
- Позже: опциональная регистрация
- При логине: localStorage мержится с серверными данными
- Бэкенд: REST API (Node/Express или Firebase)

## MVP — первый релиз

- Главная страница с карточкой Java (другие языки: "Скоро")
- Курс Java: первые 3-5 модулей с полным контентом
- Система прогресса в localStorage
- Адаптивная вёрстка (desktop + mobile)
- Без авторизации
